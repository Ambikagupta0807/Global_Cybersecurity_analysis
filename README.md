# Global_Cybersecurity_analysis
Global Cyber Crime Analysis Dashboard built using Excel and Power BI. The Excel dataset is used as the data source to create an interactive dashboard that visualizes cyber crime trends across different countries and years, helping identify patterns and high-risk regions through clear charts and insights.
📌 Project Overview

This project focuses on analyzing global cyber crime trends using data analytics and visualization tools. The goal of this project is to transform raw cyber crime data into meaningful insights that help understand patterns, affected regions, and trends over time.

The dataset was first cleaned and prepared using Python (Pandas) and SQL to ensure the data is structured, accurate, and ready for analysis. After cleaning the dataset, the data was stored in Microsoft Excel and used as the data source to build an interactive Power BI dashboard.

The final dashboard provides a clear and visual understanding of cyber crime patterns across different countries and time periods.

⚙️ Tools & Technologies Used
Python (Pandas) – Data cleaning and preprocessing
SQL – Data querying and data preparation
Microsoft Excel – Data storage and structured dataset
Power BI – Data visualization and dashboard creation
📊 Dashboard Features
Global overview of cyber crime trends
Country-wise analysis of cyber crime cases
Year-wise trend visualization
Interactive filters for better data exploration
Clear visual representation using charts, graphs, and KPIs
🧹 Data Cleaning Process

The dataset was cleaned before visualization to ensure accurate analysis. The cleaning process included:

Removing duplicate records
Handling missing values
Standardizing column names and data formats
Filtering inconsistent or irrelevant data
Querying and structuring the dataset using SQL
Data preprocessing using Python Pandas
🎯 Project Objective

The main objective of this project is to demonstrate data analysis and visualization skills by transforming raw cyber crime data into an interactive dashboard that helps users easily understand global cyber crime patterns.

📈 Outcome

This project highlights how data analytics tools like Python, SQL, Excel, and Power BI can be used together to clean, analyze, and visualize real-world data effectively.
